HTML Elements Showcase - Inline vs Block

Introduction

In HTML, elements are broadly divided into two categories: block-level and inline. Block elements take up the full width, while inline elements only take as much space as needed.

[Block Elements] [Inline Elements] [Comparison]

Block Elements

  1. Example: <p>This is a paragraph.</p>
    Explanation: The <p> tag defines a block of text that always starts on a new line.

  2. Example: <h1>This is a heading</h1>
    Explanation: Headings are block-level elements that take the full width.

  3. Example: <blockquote> "Coding is the new literacy." </blockquote>
    Explanation: Blockquote is a block container for quotations.

  4. Example: <div>This is a division or a section.</div>
    Explanation: The <div> tag is a block-level container used to group content.

  5. Example: <ul>This is an unordered list.</ul>
    Explanation: The <ul> tag defines a block of list items.

  6. Example: <ol>This is an ordered list.</ol>
    Explanation: The <ol> tag defines a block of numbered list items.

  7. Example: <li>This is a list item.</li>
    Explanation: The <li> tag defines a block-level list item.

  8. Example: <table>This is a table.</table>
    Explanation: The <table> tag defines a block-level table.

  9. Example: <header>This is a header.</header>
    Explanation: The <header> tag defines a block-level header section.

  10. Example: <form>This is a form.</form>
    Explanation: The <form> tag defines a block-level form for user input.

Inline Elements

  1. Example: <span>This is a span.</span>
    Explanation: The <span> tag is an inline container for text.

  2. Example: <a>This is a link</a>
    Explanation: The <a> tag defines a hyperlink, which is an inline element.

  3. Example: HTML stands for <abbr>HTML</abbr>. Explanation: Abbr defines an abbreviation.

  4. Example: <strong>This is strong text.</strong>
    Explanation: The <strong> tag is used to define important text.

  5. Example: <img>
    Explanation: The <img> tag is used to embed images.

  6. Example: <cite>This is a citation.</cite>
    Explanation: The <cite> tag is used to define the title of a creative work.

  7. Example: <select>HTML</select>
    Explanation: The <select> tag is used to create a drop-down list.

  8. Example: <time>This is a time element.</time>
    Explanation: The <time> tag is used to represent a specific period in time.

  9. Example: <sub>This is a subscript.</sub>
    Explanation: The <sub> tag defines subscript text.

  10. Example: <sup>This is a superscript.</sup>
    Explanation: The <sup> tag defines superscript text.
